# Blog Zone Client
    - Simple blog website powered by NodeJS - ReactJs - MongoDb 

# Installation
    - Clone the project
    - Create .env file
    - Add REACT_APP_API_URI='Blog_zone_api_uri'
    - Run
```bash
npm start
```

# Usage 
    - Blog_zone client is running on port 3000
